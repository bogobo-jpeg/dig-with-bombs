# controls

- A, D, Space - basic movement
- Q - return home
- 1, 2, 3, etc. - switch between bombs (used for purchasing also)
- RMB or Enter - place bomb
- Z - Laser shoot (LASER ONLY)
- X - Move down (LASER ONLY)
